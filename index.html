My IT Project Idea

Overview

My IT project is to develop virtual reality game. The game is a physics-based melee combat game set in a medieval time with phantasy elements. The game will primarily focus on online one on one PVP. The game will be developed using Unity. Unity is a cross platform game engine and has an XR interaction toolkit that can be used to create the VR elements of the game. The networking of the game will be done using Photon networking with Unity.

Motivation

VR as a gaming platform is growing every year (Smith 2021). It is an immersive way to play video games and has many unique experiences. VR is the perfect medium for a melee combat based game. There are many melee combat focused VR video games out, but none of them have online multiplayer. This project aims to create an online PVP melee combat focused game to fill the void in the market.

Description

The game is set in a medieval time with phantasy elements. This means the game will be using traditional medieval weapons and locations. Such weaponry will include swords, shields, axes, rapiers and many more similar weapons. Combat will take place in various medieval themed arenas. The phantasy elements of the game are in the form of spells and sorceries.

The game is to be played in virtual reality. This means the game will require a VR headset. The game is room scale VR which means it will use six depth of field tracking and the player will physically move around the world created. They will interact with every object in game using controllers that replicated inputs from hands.

The games combat will be based on physics-based interactions. The player will use the weapons by using VR controllers to simulate hands. They will be able to move their hand and squeeze on the trigger to grab items. This means they can grab a weapon and swing it themselves with their hands. They will also be able to grab and use a shield to block. Players can also use their hands to wield ranged weapons, such as a bow which requires two hands to use. Spells will be able to be selected using the controllers, and cast using movements of the hands.

Players control movement in game with a combination of the analogue stick as well as moving physically themselves. When the player is stationary, they can move the left analogue stick to move around and control the direction they are facing with the right analogue stick. The player can also use the tracking inherent with VR to move themselves within the area they have available. They allow for the player to physically dodge attacks and maneuverer during combat.

The interactions between players and objects are to be simulated as close to life as possible. This means heavy weapons cannot be used effectively with one hand and will require two hands. Weapons have real weight and cannot be swung as fast as a hand can move in the real life. Objects can be thrown and will follow a realistic trajectory. The interactions between players clashing swords, hitting each other to deal damage will make up the bread and butter of gameplay.

There will be a sandbox training mode, where the player can practice using the various weapons and sorceries at their disposal. The sole game mode will be one verses one online. In this game mode it is a simple one verse one fight between two players to the death. It will be a best of three system, where the first person to win three rounds wins. There will be online matchmaking as well as private lobbies. The two online playlists will be a casual mode and a ranked multiplayer mode.

Skills and Tools Required

The first skill required will be the ability to make games using Unity. This means creating all the assets and scenes as well as all the gameplay elements. When creating a VR game in unity, you need to add the XR interaction tool kit. This toolkit is used to enable VR features such as head tracking and controller tracking. While using Unity to create a game, I need to be able to write scripts.

To setup the multiplayer aspects of this game I will need to learn how to use photon servers in conjunction with Unity. I can use Unity’s Visual Scripting tool but learning to code in C# is ideal. To test this VR game, a VR headset is required.

Preferably I would have access to all the popular VR headsets to test the game on each platform. For the multiplayer aspects at least one other individual will need to have access to VR headsets to test the online multiplayer with me.

Outcome

The outcome of this project would be a completed online PVP VR melee combat game. The game will have working melee combat. It will closely simulate real life combat with the weapons in the game. The game will have functional online multiplayer. When completed and put on the market, it will fill the void in the VR games industry of their being no PVP VR melee combat games.
